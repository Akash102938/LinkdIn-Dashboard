LinkedIn Dashboard Redesign â€” Brief

Research & Insights

  • Key elements identified: Header (search + nav), Left sidebar (profile + quick links), Main feed (create post + activity), Right sidebar (news, people suggestions).
  • Opportunities: Reduce visual clutter, increase spacing, make news more scannable with cards, group quick links and provide collapsible behavior for small screens.

Design Goals

  1. Simplify navigation and prioritize readable content hierarchy.
  2. Improve accessibility: sufficient contrast, larger tap targets, proper focus states.
  3. Modernize news and sidebar components to improve scannability.

Key Decisions

  • Header: centered search for discoverability; compact nav with icons + labels for clarity.
  • Sidebar: profile mini-card with quick metrics; grouped quick links to reduce cognitive load.
  • Main feed: profile summary at top to encourage personal branding; create-post uses rounded input and discrete action buttons.
  • News: card layout to allow quick skimming and better visual separation from feed.

Accessibility Notes

Inputs and buttons include focus styles. Font sizes selected for legibility and ARIA labels added for core interactive controls. Contrast ratio checked for primary elements (LinkedIn blue on white meets contrast).

Next Steps

1) Move design into Figma and recreate components as variants for reusable system. 2) Add micro-interactions (hover, focus, enter post). 3) Iterate with usability testing on small group.
